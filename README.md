# NutricaoWeb
#### Este projeto é uma aplicação desenvolvida com o framework Django, permitindo o cadastro, listagem e gerenciamento de nutricionistas. A interface utiliza HTML, Django Templates e Bootstrap para um layout mais moderno, organizado e responsivo.

## Requisitos para utilização

### Para instalar, executar e editar este sistema, é necessário possuir:

#### - GitHub Desktop (para clonar o repositório)
#### - PyCharm
#### - Python 3.10 ou superior
#### - Pip (gerenciador de pacotes do Python)
#### - Django 5.0 ou superior

## Como instalar
#### 1. Abra o GitHub Desktop.
#### 2. Clique em File > Clone repository.
#### 3. Cole o link deste repositório (https://github.com/brunadelgobo-lab/nutricaoWeb.git).
#### 4. Escolha uma pasta local e selecione Clone.
#### 5. Em seguida, abra o projeto no PyCharm.
   
## Configuração do ambiente (Terminal)
### 1. No terminal integrado do PyCharm, para criar um ambiente virtual, execute:

##### python -m venv venv<br>

### 2. Ative:

#### Windows:
##### venv\Scripts\activate

#### Linux/Mac:
##### source venv/bin/activate

### 3. Instale o django:
##### pip install django

### 4. Rode as migrações do banco:
##### python manage.py migrate

### 5. Execute o servidor:
##### python manage.py runserver
